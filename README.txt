# Одежда для ручек


Приложение для просмотра материалов и их управления.


## Начало работы



1. Запустить MS SQL Server MAnagement Studio 18.8
2. Нажать "New query"
3. Переместить содержимое файла script Cloths.sql в окно запроса
4. Нажать "Выполнить", бд будет загружена вместе с данными.
5. Открыть Visual Studio 
6. Склонировать проект с git
7. Запустить проект

### Необходимые условия



1. MS SQL Server 18
2. MS SQL Management Studio 18.8
3. Visual Studio 2018
